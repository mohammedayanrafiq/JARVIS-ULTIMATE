🚀 Meet J.A.R.V.I.S — an AI assistant I built from the ground up.

Not just another chatbot wrapper. An assistant designed to think, act, and integrate — inspired by Tony Stark's onboard AI, but built with real, working tech, piece by piece.

Here's what's inside 👇

🔒 Security first
A PIN-locked entry screen with fully offline, local device authentication — no cloud dependency just to unlock it.

🧠 Core intelligence
Powered by Azure OpenAI, with a proper boot sequence, live "thinking" states while it processes, and a dedicated Memory feature so it holds context.

🧮 100% local calculator suite
31 built-in calculators across Finance, Math, and Everyday categories — Compound Interest, EMI, Quadratic Equations, Currency Conversion, and more. Zero API calls, zero hallucination risk.

🔗 App integrations
One-tap launch for Gmail, Google Drive, Docs, Sheets, Calendar, Notion, Slack, Trello, Asana, Figma, and more — JARVIS doubles as a control center for my whole workflow.

📰 AI news briefings
Ask what's new in AI & tech and get an executive-style morning briefing, generated on the spot.

🎙️ Voice mode
Wake-word activation ("Jarvis"), a live waveform listening UI, and a choice of TTS voices — with adjustable rate & pitch to match the vibe.

📷 Vision features
Camera-powered Food Scanner and a 20-second Vitals Check, running on the Gemini Vision API.

🎨 Fully themeable
Light and dark interfaces, both holding onto the same premium, minimal sci-fi feel I wanted from day one.

⚙️ Configurable core
Bring your own Azure OpenAI endpoint, API key, and deployment name, with optional live web search — built to be extensible, not locked down.

Built alongside Gaurav Keshari.

This has been months of iterating — fixing a broken API layer, redesigning the UI more than once, rebuilding features that didn't work the first time. And it's still growing — deeper automation and true agentic capabilities are next.
